# Abyss Survivors

An underwater-themed survivor game where you pilot a submarine, fight deep-sea creatures, and upgrade your ship.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173` (or the port shown in the terminal).

## Controls

- **WASD / Arrow Keys**: Move the submarine
- **Auto-fire**: The submarine automatically shoots at the nearest enemy.

## Features

- **Endless Waves**: Enemies spawn continuously.
- **Leveling System**: Collect XP orbs dropped by enemies to level up.
- **Upgrades**: Choose from 3 random upgrades every time you level up (Attack Speed, Damage, Movement Speed, Health).
- **Enemies**:
  - **Fish**: Basic chasers.
  - **Jellyfish**: Pulsing movement, higher HP.

## Tech Stack

- TypeScript
- Vite
- HTML5 Canvas
